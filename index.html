My Work

RNN with LSTM cells

Stacked BiRNN model that processes text data using pretrained word embeddings and a bidirectional LSTM architecture, capturing both forward and backward dependencies for improved context understanding. It employs global max pooling over time steps to extract the most relevant features before classification, ensuring efficient sentiment analysis with early stopping and learning rate scheduling for optimal training.

Random Forest with ID3

Random Forest implementation that builds an ensemble of ID3 decision trees, each trained on bootstrapped subsets of the data with randomly selected features per split. The final prediction is determined by majority voting, ensuring robustness and reducing overfitting while maintaining efficiency in handling high-dimensional feature spaces

Logistic Regression

Logistic Regression model that uses sparse feature representations and stochastic gradient descent (SGD) with L2 regularization to classify sentiment. You incorporated dynamic learning rate decay, a structured evaluation pipeline, and a learning curve generator to optimize and assess model performance effectively

Naive Bayes

Naive Bayes Classifier efficiently processes tokenized text by converting it into binary vectors and applying Bernoulli Naive Bayes with Laplace smoothing for sentiment analysis. Implemented a structured evaluation pipeline, including precision, recall, F1-score calculations, and a learning curve generator to assess model performance across different training sizes

Elasticsearch TF-IDF Retrieval Baseline

Implemented a classic VSM/TF-IDF retrieval baseline on top of Elasticsearch. The pipeline ingests a CSV corpus, builds an index, executes batch queries, and exports TREC-formatted run files (k=20/30/50) for evaluation with tools like trec_eval

Genetic Algorithm

The genetic algorithm optimizes high school timetables by evolving schedules through selection, crossover, and mutation. It ensures constraints like balanced teaching hours, no gaps, and teacher availability are met, efficiently generating practical and near-optimal solutions.

Distributed Efood Systen

A Java-based distributed e-food platform using Master/Worker architecture with replication, failover, and a Reducer server for MapReduce-style aggregation. Demonstrates distributed systems, socket RPC, replication, and fault tolerance with client and manager console apps.

Backward Chaining Deduction System

Forward Chaining System
